# Privacy Policy for AI Content Extractor

**Last Updated: January 2025**

## Overview

AI Content Extractor ("the Extension") is committed to protecting your privacy. This privacy policy explains how our browser extension handles your data.

## Data Collection

### What We DON'T Collect
- Personal information (name, email, etc.)
- Browsing history
- Cookies or tracking data
- User analytics
- Content you extract

### What We DO Process (Locally Only)
- Web page content for extraction (processed locally in your browser)
- Extension settings (stored locally using Chrome Storage API)
- Temporary clipboard data (for copy functionality)

## Data Storage

All data is stored locally on your device:
- **Extension Settings**: Stored using Chrome's Storage API
- **Extracted Content**: Temporarily held in browser memory, not saved
- **No External Servers**: We do not send any data to external servers

## Permissions Usage

The extension requires certain permissions to function:

- **activeTab**: To read content from the current webpage
- **storage**: To save your preferences locally
- **clipboardWrite**: To copy extracted content to clipboard
- **contextMenus**: To add right-click menu options
- **notifications**: To show extraction status messages
- **host_permissions (<all_urls>)**: To extract content from any website you visit

## Third-Party Services

This extension does NOT use any third-party services, analytics, or tracking.

## Data Security

- All processing happens locally in your browser
- No data transmission over the internet
- No external API calls
- No user tracking or profiling

## Children's Privacy

This extension does not knowingly collect any information from children under 13 years of age.
