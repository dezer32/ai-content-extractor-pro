# AI Content Extractor для Brave/Chrome

Расширение для браузера, которое извлекает читаемый контент с веб-страниц и форматирует его для использования в AI чатах.

## Возможности

- **Умное извлечение контента** - автоматически определяет основной контент страницы
- **Очистка от мусора** - удаляет навигацию, рекламу, виджеты и другие нерелевантные элементы
- **Markdown форматирование** - конвертирует HTML в читаемый Markdown
- **Настраиваемые опции**:
  - Включение/исключение изображений
  - Включение/исключение ссылок
  - Выбор формата вывода (Markdown или простой текст)
- **Статистика контента** - показывает количество слов, символов и время чтения
- **Контекстное меню** - быстрое извлечение выделенного текста или всей страницы
- **Горячие клавиши** - быстрый доступ к функциям

## Установка

### Шаг 1: Загрузка файлов

Создайте новую папку для расширения и добавьте в неё следующие файлы:

1. `manifest.json` - конфигурация расширения
2. `popup.html` - интерфейс popup окна
3. `popup.js` - логика popup окна
4. `content.js` - скрипт извлечения контента
5. `background.js` - фоновый service worker
6. `offscreen.html` - вспомогательный файл для clipboard

### Шаг 2: Создание иконок

Создайте иконки для расширения в формате PNG:
- `icon-16.png` (16x16 px)
- `icon-32.png` (32x32 px)
- `icon-48.png` (48x48 px)
- `icon-128.png` (128x128 px)

Можно использовать простую иконку с буквами "AI" или символом извлечения текста.

### Шаг 3: Установка в браузер

1. Откройте Brave/Chrome
2. Перейдите в `brave://extensions/` или `chrome://extensions/`
3. Включите "Режим разработчика" (Developer mode) в правом верхнем углу
4. Нажмите "Загрузить распакованное расширение" (Load unpacked)
5. Выберите папку с файлами расширения
6. Расширение появится в списке и будет готово к использованию

## Использование

### Основной способ:

1. Кликните на иконку расширения в панели браузера
2. Нажмите кнопку "Extract" для извлечения контента
3. Просмотрите preview и статистику
4. Нажмите "Copy" для копирования в буфер обмена
5. Вставьте контент в ваш AI чат

### Через контекстное меню:

1. Выделите текст на странице
2. Правый клик → "Extract selected text for AI"
3. Текст будет отформатирован и скопирован

### Настройки:

- **Include Images** - включать описания изображений
- **Include Links** - сохранять ссылки в формате Markdown
- **Markdown Format** - использовать Markdown форматирование

## Алгоритм извлечения

Расширение использует многоуровневый подход для определения основного контента:

1. **Поиск семантических тегов** - `<article>`, `<main>`, `[role="main"]`
2. **Анализ классов и ID** - поиск индикаторов контента (article, content, post, etc.)
3. **Статистический анализ** - определение блока с наибольшим количеством текста
4. **Очистка контента**:
   - Удаление скриптов, стилей, навигации
   - Фильтрация рекламных блоков
   - Удаление скрытых элементов
5. **Форматирование**:
   - Конвертация в Markdown
   - Сохранение структуры документа
   - Оптимизация для AI обработки

### Для анализа данных:

Расширение сохраняет таблицы в Markdown формате, что удобно для анализа структурированных данных.

### Для исследований:

Извлекайте контент из множества источников с сохранением ссылок и метаданных.

## Безопасность и приватность

- Расширение работает локально в браузере
- Не отправляет данные на внешние серверы
- Не сохраняет историю извлечений
- Не требует регистрации или авторизации
- Открытый исходный код

## Горячие клавиши

Вы можете настроить горячие клавиши в `brave://extensions/shortcuts`:

- **Быстрое извлечение** - по умолчанию не назначено

## Известные ограничения

1. Некоторые SPA (Single Page Applications) могут требовать дополнительного времени для загрузки контента
2. Контент за paywall или требующий авторизации может быть недоступен
3. Динамически загружаемый контент (infinite scroll) извлекается только видимая часть
4. PDF и другие встроенные документы не обрабатываются

## Поддержка

При возникновении проблем:

1. Убедитесь, что страница полностью загружена
2. Попробуйте обновить страницу и повторить извлечение
3. Проверьте консоль браузера на наличие ошибок (F12)
4. Попробуйте отключить другие расширения, которые могут конфликтовать
